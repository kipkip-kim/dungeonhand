## 2. Changes Detail

### 2.1 P0: scaleMonsterHp/Atk 이중 스케일링 제거

**원인**: MONSTERS 테이블에 이미 층별 HP/ATK가 설정되어 있으나 (고블린 36hp → 드래곤로드 260hp), `scaleMonsterHp(baseHp, fl)` 함수가 `baseHp * (1 + (fl-1)*0.45)` 추가 배율을 적용.

**수정**:
- `scaleMonsterHp`, `scaleMonsterAtk` 함수 삭제 (DungeonHand_v3.jsx)
- `beginBattle`: `scaleMonsterHp(m.hp, fl)` → `m.hp`, `scaleMonsterAtk(m.atk, fl)` → `m.atk`
- 캠프 기습: 동일 패턴 적용

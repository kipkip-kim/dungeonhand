# Plan: 밸런스 논의

## Executive Summary

| 관점 | 내용 |
|------|------|
| **문제** | 몬스터 HP 이중 스케일링(드래곤로드 HP 728), 일부 유물/스킬 효율 불균형 |
| **해결** | scaleMonsterHp/Atk 제거 + 약한 유물/스킬 상향 + 강한 스킬 하향 |
| **기능/UX 효과** | 5층 보스 의도된 난이도(HP 260)로 복원, 유물/스킬 선택 다양성 증가 |
| **핵심 가치** | 밸런스 정상화로 게임 클리어율 적정화 + 빌드 다양성 확보 |

---

## 1. 배경

Game Balance 에이전트 분석 결과, 다음 밸런스 이슈가 확인됨:

- **[P0 치명]** `scaleMonsterHp/Atk` 이중 스케일링 — MONSTERS 테이블에 이미 층별 HP/ATK가 반영되어 있으나, 전투 시작 시 `(fl-1)*0.45` (HP) / `(fl-1)*0.10` (ATK) 추가 곱셈 적용
  - 1층: 정상 (배율 1.0)
  - 3층: HP x1.9, ATK x1.2
  - 5층: HP x2.8, ATK x1.4 → **드래곤로드 HP 728, ATK 28** (의도: 260, 20)
- **[P1]** 유물/스킬 효율 불균형
- **[P2]** 일부 키워드/스킬 약세
- **[P3]** 경제 미세 조정

---

## 2. 변경 계획

### P0: 몬스터 HP 이중 스케일링 제거 (필수)

| 항목 | 현재 | 변경 |
|------|------|------|
| `scaleMonsterHp(baseHp, fl)` | `baseHp * (1 + (fl-1)*0.45)` | 삭제 → `baseHp` 직접 사용 |
| `scaleMonsterAtk(baseAtk, fl)` | `baseAtk * (1 + (fl-1)*0.10)` | 삭제 → `baseAtk` 직접 사용 |

**영향 범위**: `beginBattle` (line 326-327), 캠프 기습 (line 1151-1152)

**검증**: 5층 드래곤로드 HP=260, ATK=20 확인

### P1: 유물/스킬 효율 조정

| 항목 | 현재 | 변경 | 사유 |
|------|------|------|------|
| 가시 갑옷 (thorn) 반사 | 3 | 5 | 피격당 3은 무의미, 5로 상향 |
| 약탈 (loot) 골드 보너스 | +3/레벨 | +2/레벨 | lv2 시 +6G → 기본 골드(4~8G)의 75~150%, 과도함 |
| 공명 (resonance) 배율 | +0.5 | +0.8 | 같은 문양 2장+ 조건 대비 +0.5 약함 |

### P2: 약세 스킬 상향

| 항목 | 현재 | 변경 | 사유 |
|------|------|------|------|
| 은신 (stealth) 회피 보너스 | +5%/레벨 | +7%/레벨 | critMastery(+10%/lv) 대비 투자 효율 낮음 |
| 연쇄의 고리 유물 배율 | +2 | +3 | 티어2 유물 중 가장 약함 (스트레이트 x4+2=6 vs 감정사 다수 카드 배율) |

### P3: 경제 미세 조정

| 항목 | 현재 | 변경 | 사유 |
|------|------|------|------|
| 캠프 상인 판매가 | 카드등급 × 2G | 카드등급 × 3G | 등급3 카드 6G → 9G, 판매 선택지 의미 있게 |

# Design: 밸런스 논의

## Executive Summary

| 관점 | 내용 |
|------|------|
| **문제** | 몬스터 HP 이중 스케일링(드래곤로드 HP 728), 일부 유물/스킬 효율 불균형 |
| **해결** | scaleMonsterHp/Atk 제거 + 데이터 상수 조정 4건 |
| **기능/UX 효과** | 5층 보스 의도된 난이도 복원, 빌드 다양성 증가 |
| **핵심 가치** | 밸런스 정상화 + 선택지 간 효율 균등화 |

---

## 1. 변경 상세 설계

### 1-1. scaleMonsterHp/Atk 이중 스케일링 제거 [P0]

**현재 코드** (`DungeonHand_v3.jsx:219-220`):
```javascript
function scaleMonsterHp(baseHp, fl) { return Math.floor(baseHp * (1 + (fl - 1) * 0.45)); }
function scaleMonsterAtk(baseAtk, fl) { return Math.floor(baseAtk * (1 + (fl - 1) * 0.1)); }
```

**문제**: MONSTERS 테이블에 이미 층별 HP/ATK 반영됨 (고블린 36hp → 드래곤로드 260hp). 스케일링 함수가 추가 배율을 적용하여 5층 HP가 2.8배가 됨.

**변경**: 두 함수 삭제, 호출부에서 `m.hp` / `m.atk` 직접 사용.

**호출부 (2곳)**:

| 위치 | 현재 | 변경 |
|------|------|------|
| `beginBattle` (line 326-327) | `scaleMonsterHp(m.hp, fl)` / `scaleMonsterAtk(m.atk, fl)` | `m.hp` / `m.atk` |
| 캠프 기습 (line 1151-1152) | `scaleMonsterHp(am.hp, floor)` / `scaleMonsterAtk(am.atk, floor)` | `am.hp` / `am.atk` |

**검증 기준**:
- 1층 고블린: HP=36, ATK=6 (변화 없음, 1층은 배율 1.0이었음)
- 5층 드래곤로드: HP=260, ATK=20 (기존 728/28에서 복원)

---

### 1-2. 가시 갑옷 반사 상향 [P1]

**파일**: `data.js` RELICS 배열

**현재**: `{ id: "thorn", ..., eff: { type: "thorns", val: 3 } }`
**변경**: `val: 3` → `val: 5`, desc "피격 시 적에게 3 반사" → "피격 시 적에게 5 반사"

**사유**: 몬스터 HP 36~260 대비 피격당 3 반사는 기여도 1~8%. 5로 상향하면 2~14%, 유의미한 선택지.

---

### 1-3. 약탈 골드 보너스 하향 [P1]

**파일**: `DungeonHand_v3.jsx` line 897

**현재**: `upgradeLevels.loot * 3`
**변경**: `upgradeLevels.loot * 2`

**파일**: `data.js` SKILL_TREES loot 노드
**현재**: `desc: "전투 골드 +3"`
**변경**: `desc: "전투 골드 +2"`

**사유**: lv2 기준 +6G는 기본 골드(4~8G)의 75~150%. +4G(50~100%)가 강하되 과도하지 않음.

---

### 1-4. 공명 키워드 배율 상향 [P1]

**파일**: `utils.js` calcDamage 내 resonance 분기 (line 189-193)

**현재**: `mult += 0.5`
**변경**: `mult += 0.8`

**파일**: `data.js` KEYWORDS resonance
**현재**: `desc: "같은 문양 2장+ 시 배율 +0.5"`
**변경**: `desc: "같은 문양 2장+ 시 배율 +0.8"`

**사유**: 같은 문양 2장 이상 조건은 덱 구성에 제약. +0.5는 기세(무조건 +0.5)와 동일하여 조건부 효과로서 약함.

---

### 1-5. 은신 회피 보너스 상향 [P2]

**파일**: `DungeonHand_v3.jsx` buildPState (line 265)

**현재**: `stealthBonus: upgradeLevels.stealth * 5`
**변경**: `stealthBonus: upgradeLevels.stealth * 7`

**파일**: `data.js` SKILL_TREES stealth 노드
**현재**: `desc: "기본 회피 +5%"`
**변경**: `desc: "기본 회피 +7%"`

**사유**: critMastery는 +10%/lv로 직접적인 DPS 증가. 은신은 +5%/lv 방어적 효과로 투자 효율이 낮음. +7%로 올려 방어 빌드 유인 제공.

---

### 1-6. 연쇄의 고리 유물 배율 상향 [P2]

**파일**: `data.js` RELICS chain

**현재**: `{ id: "chain", ..., desc: "스트레이트 배율 +2", ..., eff: { type: "handAdd", hand: "스트레이트", val: 2 } }`
**변경**: `val: 2` → `val: 3`, desc "스트레이트 배율 +2" → "스트레이트 배율 +3"

**사유**: 티어2 유물 비교
- 루비 반지: 🔺카드 x2 (안정적, 매번 적용)
- 감정사의 눈: 등급5↑ 카드당 배율+2 (높은 천장)
- 연쇄의 고리: 스트레이트 +2 (조건부, 스트레이트3 x4→x6) — 조건 대비 보상 약함. +3으로 올려 x4→x7.

## 3. 영향 분석

### 변경 없는 파일
- `components.jsx` — 유물/키워드 표시는 data 참조이므로 자동 반영
- `screens/*.jsx` — VillageScreen의 스킬 트리 desc는 data.js 참조
- `audio.js`, `styles.js` — 무관

### 주의 사항
- `DungeonHand_v3_portrait.jsx` — **절대 수정 금지** (원본 백업)
- `DungeonHand_v3_iphone14pro.jsx`, `DungeonHand_v3_test.jsx` — 별도 파일, 이번 변경 대상 아님
- `rollEnemyDmg` 함수는 유지 (ATK 기반 랜덤이므로 스케일링과 무관)

### 밸런스 영향 요약

| 항목 | Before | After | 변동 |
|------|--------|-------|------|
| 5층 드래곤로드 HP | 728 | 260 | -64% (의도된 값 복원) |
| 5층 드래곤로드 ATK | 28 | 20 | -29% |
| 3층 대마법사 HP | 271 | 143 | -47% |
| 가시 갑옷 피격당 반사 | 3 | 5 | +67% |
| 약탈 lv2 골드 보너스 | +6G | +4G | -33% |
| 공명 배율 | +0.5 | +0.8 | +60% |
| 은신 lv2 회피 보너스 | +10% | +14% | +40% |
| 연쇄의 고리 배율 | +2 | +3 | +50% |
